PART1--初识JavaScript

JavaScript是什么？

通常对javascript的定义有多种多样，但是总的来说，它是一个面向WEB应用 的编程语言，HTML是一个标记语言，javascript才是作为WEB开发工程师面临 学习的第一门编程语言，学习编程语言，需要我们动用大脑，尽情的发挥我们的逻辑 思维能力。由于它的重要性，目前各大浏览器都支持JavaScript运行。

另外，JavaScript还被赋予以下标签：

脚本语言，意思是JavaScript单独存在是没有意义的，它需要嵌入在HTML页面中， 在浏览器中正常运行，才能发挥他的作用；

轻量级语言，意思是这门语言相对C/C++,Java等其他编程语言来说，语法规则更少 更简单，更容易学习；

弱类型语言，意思是不同于C/C++,Java，它的变量没有明确的数据类型的概念， 在运行过程中是可以变换的；

动态语言，与弱类型语言类似，意思是它的变量数据类型不是定义是明确指定，一成 不变的，是可以改变的

JavaScript作用

一句话简单的概括，它使网页“活”起来了，不再是一个安静的美男子， 有一些动态的画面以及和用户的交互功能等等。

具体来说就是：

① 能够动态改变网页的内容（包括结构和样式以及文本内容等）

② 能够与用户进行交互能够对HTML中的表单进行验证等

③ 能够与用户进行交互

④ 能够控制浏览器的前进后退等

PART3--JavaScript函数

函数

函数定义关键字：function

函数定义语法:

方法一： function 函数名(a,b) {...} ，其中函数名(a,b) 称为函数头， {} 称为函数体。

方法二： var 变量名 = function() {}; 这时候函数本质是匿名函数，有人也习惯把变量名看做=右边匿名函数的名字。

根据()中是否带参数分为：有参函数、无参函数

参数根据使用场景分为：

函数实参：调用时候使用的实际变量

函数形参：定义时候时候的临时参数名

arguments 对象:

定义：类似数组，可以获取函数执行时的实际参数情况。

掌握：arguments.length, argument[i]

参数规则：

1.实参个数小于形参，会出现undefined

函数返回值：return关键字，函数表达式的值就是返回值，没有return就是undefined， 不带任何值的return就表示退出函数，后面语句不执行

函数返回值作用：把函数表达式转化为一个变量，可以在函数外使用。

① 可以用作另一个函数参数，getAge(getName(23));

解释：有一个函数getName(id) 可以根据学生id获取学生的姓名并return 返回， 然后这值又可以作为另一个函数getAge(name)的参数来使用。

② 赋值给某个变量，var result = getAge(“Lily”);

③ 赋值给某个对象方法属性， var obj = { "age"：12， "getAge": function() { Return this.age; } };

④ 用来设置文档内容，document.write(getName(12));

⑤ 表达式的一部分，alert(“Hello” + getName(12));

函数组成：关键字，函数名(标识符)，函数参数(变量)，函数体(代码块)

函数本质：由事件驱动(dom阶段讲)的或者当它被调用时执行的可重复使用的代码块。

函数提升：提升（Hoisting）应用在变量的声明与函数的声明。是指实际使用变量或者调用函数在声明之前进行。

函数作用：代码复用、可移植性、逻辑清晰

函数触发执行：事件驱动、人为设置代码调用

函数调用：先定义随后（通过函数名）调用，立即调用(匿名函数)

函数变量作用域：局部变量，全部变量

BOM

BOM(Browser Object Model)定义:

(浏览器对象模型)浏览器提供给我们获取/操作浏览器相关属性的API。通俗的讲， 就是暴露给我们一些方法，通过调用这些函数，我们可以获取浏览器的一些属性， 并且可以修改这些属性。BOM通常操作对象是除了html文档之外的东西，比如地址栏， 控制台，访问记录历史栈，还有我们的屏幕，浏览器自身等。

location对象

通常用来获取浏览器地址栏的相关信息。

history对象

通常用来控制浏览器前进后退访问曾经浏览过的页面。history对象记录了 用户曾经浏览过的页面，并可以实现浏览器前进与后退相似的导航功能。

navigator对象

screen对象

screen.availWidth 窗口可以使用的屏幕高度，单位px,不包括Windows任务栏

screen.availHeight 窗口可以使用的屏幕宽度，单位px,不包括Windows任务栏

screen.width 返回屏幕的总宽度

screen.height 返回屏幕的总高度

全局对象window

window对象的全局属性

用来确定浏览器窗口的尺寸（浏览器的视口，不包括工具栏和滚动条）：

① window.innerHeight - 浏览器窗口的内部高度

② window.innerWidth - 浏览器窗口的内部宽度对于 Internet Explorer 8、7、6、5

③ document.documentElement.clientHeight

④ document.documentElement.clientWidth

⑤ document.body.clientHeight

⑥ document.body.clientWidth

各种浏览器兼容性问题，通常的做法是：

//获取宽度

var w=window.innerWidth|| document.documentElement.clientWidth|| document.body.clientWidth;

//获取高度

var h=window.innerHeight|| document.documentElement.clientHeight|| document.body.clientHeight;

PART6--JavaScript高级

预解析

Js引擎有预解析行为，预解析跟变量提升常结合说明，也就是说在js中 我们可以在一个变量声明前或者函数定义前使用而不会报错：is not defined， 而是可以正确的运行下去，这种行为本质上可以打比方为在js函数真正运行之前， js引擎会全局搜索一遍，把所有的变量和函数都放在一个白名单里，同时把他们的值 初始化为undefined，对于函数会存放函数的地址，预解析完成后，函数会真正的 逐行执行代码，然后到达使用某个变量的时候，会在白名单中检查一下是否具有这个名字 配置，如果么有找到，说明没有声明，报错is not defined，如果找到了，则不会报错， 具体的值分两种情况：

1）代码还没有运行到变量赋值的地方，所以这个时候变量的值还是预解析时候的undefined；

2）已经在使用之前定义了变量的值了，这个时候会使用真正的值。

变量提升

在JS中，就是把定义在后面的东东（变量或函数）提升到前面中定义。 变量提升只是提升变量的声明，并不会把赋值也提升上来。

在js分为变量提升和函数提升：

变量提升，很简单，就是把变量提升提到函数的top的地方。 我么需要说明的是，变量提升 只是提升变量的声明，并不会把赋值也提升上来。

函数提升是把整个函数都提到前面去。 对于函数，一种是函数表达式， 另外一种是函数声明方式。我们需要重点注意的是，只有函数声明形式才能被提升。

闭包

在js中，每个函数在定义的时候（不是调用运行的时候）会创建一个属于自己的作用域。 而作用域可能有嵌套关系，就是说定义f1的时候在f1内部定义了f2, 这个时候f1可以 访问自己的变量还有上级全局作用域的东西，而f2可以访问自己的变量，还能访问f1中 的变量，也能访问全局作用域的变量，这就是作用域链。作用域的原则是只能向上访问比 自己大的作用域的东西，不能向下访问比自己小的作用域的内容。

总结，闭包就是创建的时候某个函数可以访问除自身之外的其他作用域的变量和函数， 调用运行的时候，不论在哪里，都可以访问声明时候可以访问的作用域。
