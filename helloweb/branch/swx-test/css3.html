  • 当在@keyframe创建动画，把它绑定到一个选择器，否则动画不会有任何效果。 指定至少这两个CSS3的动画属性绑定向一个选择器： • 规定动画的名称 • 规定动画的时长 动画是使元素从一种样式逐渐变化为另一种样式的效果。 您可以改变任意多的样式任意多的次数。 请用百分比来规定变化发生的时间，或用关键词 "from" 和 "to"，等同于 0% 和 100%。 0% 是动画的开始，100% 是动画的完成。
  • 自适应布局的一个方法：针对不同媒体类型可以定制不同的样式规则。通常是说媒体的屏幕宽度和高度划分。Css3的媒体查询更加完善，不仅可以检测出可视窗口的宽高、分辨率，还可以用来检测屏幕朝向，也就是移动端。 CSS3的媒体查询可以设备的种类以及各种功能特性实现展示不同的样式。可以形象的认为媒体查询就是对浏览器发出一个疑问，浏览器选择是，那么就采纳对应的样式，否则忽略该样式设置。早在css2.0阶段使用了link标签引入样式文件的时候，就有一个media属性，可以设定设备类型.
  • CSS3引入一种弹性盒模型技术，关于某个盒模型的分布和周边可用空间的处理。利用这种模型可以建立自适应浏览器窗口的布局以及自适应大体的弹性布局。 弹性盒模型可以规定盒子的顺序，要设置为弹性盒模型，只要设置display: box 或inline-box 多列布局各列宽度必须相等，列和列之间的宽度也是一样的。适合文章内容排版，不适合于网页各个元素的布局，这种就适合用弹性盒模型。 常见的三个div通过float:left 并列显示，然后左边列的内容比较少，中间比较多，右边比较少的时候，三块底部无法对齐，必须设置一个固定高度解决或者其他方式，但是如果使用弹性盒模型so easy。把三个浮动div外层的盒子设置为弹性盒模型，display: box
